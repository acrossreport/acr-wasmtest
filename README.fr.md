# ACR WASM Serveur de test

[English](README.md) | [日本語](README.ja.md) | Français

Serveur de previsualisation locale pour ACR (Across Report Renderer).

## Environnement requis

- Google Chrome
- Extension Chrome ACR (gratuit)
- Inscription: https://licensvr.acrossreport.com

## Utilisation

```
./acr-preview
acr-preview.exe
```

Le navigateur ouvre http://localhost:7878 automatiquement.
