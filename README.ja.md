# ACR WASM テストサーバ

[English](README.md) | 日本語 | [Français](README.fr.md)

ACR（Across Report Renderer）のローカルプレビューサーバです。
Chrome拡張とWASMエンジンを使って、ブラウザ上で帳票をPNG形式で表示します。

## 必要な環境

- Google Chrome
- ACR Chrome拡張（無償）
- ライセンス登録: https://licensvr.acrossreport.com

## 使い方

```
./acr-preview
acr-preview.exe
```

ブラウザが自動で開き http://localhost:7878 にアクセスします。
